Configurando um usuário

A primeira coisa que você deve fazer quando instalar o Git é definir o seu nome de usuário e endereço de e-mail.

git config --global user.name "Seu Nome"

git config --global user.email seu.email@email.com

Criando um novo repositório

vá a pasta do projeto e use o comando git init para inicializar um novo repositório:

git init

Clonando um repositório

Pra criar uma cópia de trabalho deum repositorio remoto para um repositorio local

git clone <URL>

quando usar um servidor remoto, seu comando será

git clone usuário@servidor:/caminho/para/o/repositório

Enviando arquivos

Esse comando irá adicionar todos os arquivos novos e/ou modificados ao repositório. O arquivo passa de Working Directory para o Staging.

git add.

Esse comando irá adicionar o arquivo em específico ao repositório

git add <Nome do arquivo>

adicionar & confirmar

Mover um arquivo do Staging para o Repository

git commit -m "comentários das alterações"
