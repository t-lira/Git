Guia básico - Git

Configurando um usuário

A primeira coisa que você deve fazer quando instalar o Git é definir o seu nome de usuário e endereço de e-mail.

git config --global user.name "Seu Nome"

git config --global user.email seu.email@email.com

Criando um repositório

Criando um novo repositório local

git init

Clonando um repositorio remoto já existente

git clone <URL>

quando usar um servidor remoto, seu comando será

git clone usuário@servidor:/caminho/para/o/repositório

Adicionar & Confirmar

Esse comando irá adicionar todos os arquivos novos e/ou modificados a Staging .

git add.

Esse comando irá adicionar um arquivo em específico a Staging .

git add <Nome do arquivo>

Mover um arquivo da Staging para o Local Repository

git commit -m "comentários das alterações"

Observe seu repositório

O comando lhe mostra em qual branch você se encontra.

git status

Mostra as linhas exatas que foram adicionadas e removidas

git diff

Enviando alterações

Mover um arquivo do Local Remote para o Remote Repository

git push
