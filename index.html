Guia básico - Git

Configurando um usuário

Este comando define globalmente o nome do usuário que será associado aos commits feitos no Git.

git config --global user.name "Seu Nome"

Este comando define globalmente o email do usuário que será associado aos commits feitos no Git.

git config --global user.email seu.email@email.com

Verificar todas as informações de configuração:

git config --list

git config --global -e

Criando um repositório

Inicializa um novo repositório Git localmente no diretório atual.

git init

Clona um repositorio remoto já existente, para o diretório local.

git clone <URL>

Clonar um repositório remoto usando um servidor remoto.

git clone usuário@servidor: <URL>

Adicionar & Confirmar

Esse comando irá adicionar todos os arquivos novos e/ou modificados ao Staging .

git add .

Esse comando irá adicionar um arquivo em específico ao Staging .

git add <Nome do arquivo>

Mover um arquivo do Staging para o Repositório local

git commit -m "comentários das alterações"

Observe seu repositório

Exibe o estado atual do repositório, incluindo arquivos modificados, adicionados e confirmados.

git status

Mostra as linhas exatas que foram adicionadas ou removidas.

git diff

Exibe o histórico de commits.

git log

Este comando exibirá cada commit em uma única linha, com apenas o hash do commit e a mensagem.

git log --oneline

Enviando alterações

Mover um arquivo do repositório local para, o repositório remoto

git push

Criar/Mudar/Excluir uma Branch

Para criar uma nova branch em um repositório Git

git branch minha-nova-feature

Para entrar na nova branch minha-nova-feature:

git checkout minha-nova-feature

Para retornar para a branch master/Main:

git checkout master

Para remover uma branch no Git:

git branch -d minha-nova-feature

Mesclar uma branch em outra

Primeiro, mude para a branch de destino:

git checkout main

Em seguida, mescle a branch "minha-nova-feature" na branch atual:

git merge minha-nova-feature

Disponibilizar uma branch para outros colaboradores em um repositório Git remoto:

git push origin minha-nova-feature

Atualizar

obter as últimas atualizações de um repositório remoto

git pull
